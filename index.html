Opaque VDI

Sporting an easy to use multi-touch interface, and capable of connecting graphically to virtual machines running under oVirt, RHEV, and Proxmox either directly for sufficiently recent oVirt/RHEV installations or through a console.vv file for all virtualization platforms. Also supports multiple keyboard layouts.

Available on Google Play and Blackberry World:

https://play.google.com/store/apps/details?id=com.undatech.opaque

http://appworld.blackberry.com/webstore/content/46004889

bVNC

A fast and stable VNC client with an intuitive multi-touch interface and an unrivalled choice of security modes including VeNCrypt, SSH tunneling, and AnonTLS. Supports Mac OS X ARD, automatic X-session spawning with x11vnc, and pretty much every VNC server on the planet.

Available on Google Play and Blackberry World:

https://play.google.com/store/apps/details?id=com.iiordanov.bVNC

https://appworld.blackberry.com/webstore/content/91989

aRDP

A fast and stable RDP client with the same multi-touch interface as bVNC. Supports everything the FreeRDP library offers, but also SSH tunneling.

Available on Google Play and Blackberry World:

https://play.google.com/store/apps/details?id=com.iiordanov.aRDP

https://appworld.blackberry.com/webstore/content/46003889

aSPICE

A fast and stable SPICE client with the same multi-touch interface as bVNC. Supports everything libspice offers, but also SSH tunneling.

Available on Google Play and Blackberry World:

https://play.google.com/store/apps/details?id=com.iiordanov.aSPICE

https://appworld.blackberry.com/webstore/content/46004887
